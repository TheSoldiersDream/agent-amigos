# Amigos System Ops (Local + Web)

## Objective

Prove Agent Amigos can operate across Windows, local tools, and web workflows.

## Tasks

- Confirm backend and frontend servers are running.
- Validate OpenWork status and list sessions.
- Check system memory and CPU usage.
- Verify access to PowerShell and CLI tools.
- Open a web search and capture top 3 results for a query.
- Summarize findings and log next actions.

## Deliverables

- Short health report with timestamps.
- Next steps list for any failures.
