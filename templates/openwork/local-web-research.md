# Local + Web Research Sprint

## Objective

Run a focused research task using local files and web sources.

## Tasks

- Define the research question and desired output.
- Search the web for 3 credible sources.
- Extract key points from each source.
- Compare against any relevant local notes or files.
- Produce a concise summary and recommendations.

## Deliverables

- Source list with links.
- Summary paragraph.
- Actionable recommendations.
