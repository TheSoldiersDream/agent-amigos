# OpenWork Agentic Workflow Test

Use this template to validate agentic behavior in OpenWork.

## Project

**Title:** Can Agent Amigos Autonomously Research, Reason, and Report?

## Objectives

- Demonstrate task decomposition and agent roles.
- Run sequential + parallel reasoning.
- Show tool awareness and intermediate outputs.
- Validate work and report outcomes.

## Workflow Steps

- Define what an agentic workflow system is in plain English.
- List 3 traits that separate agentic systems from standard chatbots.
- Create a simple 5-step workflow for complex problem solving.
- Validate the definition and workflow for gaps or vagueness.
- Produce an executive summary and self-assessment.

## Output Format

- Workflow Plan
- Agent Assignments
- Intermediate Agent Outputs
- Validation Review
- Final Executive Summary
- System Self-Assessment

## Success Criteria

- Clear delegation and intermediate outputs.
- Self-validation included.
- Final pass/fail statement.
